# Find Content in the Content Calendar

Use the date range and any controls available in Content Calendar to narrow the items you need to review.

## Find an item

1. Check that the correct month or date range is selected.
2. Use available dates, filters, statuses, client selectors, or search to narrow the visible items.
3. Clear filters when you need to return to a broader view.
4. Check another workspace or client context when appropriate.

## If you cannot find an item

Calendar visibility varies by account and permissions. Confirm the selected date range, filters, and current context before contacting support.
