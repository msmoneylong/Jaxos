# Navigate the Content Calendar

Use Content Calendar to review the planned or scheduled content visible in the selected context.

## Steps

<Steps>
  <Step>
    Open **Content Calendar**.

    **Expected result:** The calendar opens with the content visible to your account.
  </Step>
  <Step>
    Confirm the correct workspace or client context.

    **Expected result:** You are viewing the intended set of calendar content.
  </Step>
  <Step>
    Choose the available date range or calendar view.

    **Expected result:** The calendar displays content for the selected view or period.
  </Step>
  <Step>
    Review the visible content items.

    **Expected result:** You can identify the dates and statuses currently shown.
  </Step>
  <Step>
    Open an item for more details when that option is available.

    **Expected result:** You can review the item details shown in JaxOS.
  </Step>
  <Step>
    Use only the actions shown for your account.

    **Expected result:** Any available action applies only when JaxOS offers it for that item and account.
  </Step>
</Steps>
