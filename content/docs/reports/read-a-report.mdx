# Read a Report

Start with the report summary, then confirm the context for the information shown.

## Review a report carefully

1. Check the report title and subject.
2. Confirm the displayed reporting period.
3. Review the labels and metric definitions shown in the report.
4. Distinguish observations, recommendations, plans, and measured results.
5. Avoid assumptions when a data source is not identified.
