# Use a Prompt Pack

Use a Prompt Pack to work through guided prompts in its available modules. The questions and actions shown can vary by prompt, account, and current product release.

<Steps>
  <Step>
    Open the Vault.
  </Step>
  <Step>
    Select an available Prompt Pack.
  </Step>
  <Step>
    Choose a module.
  </Step>
  <Step>
    Open a prompt.
  </Step>
  <Step>
    Provide the requested business or campaign details.
  </Step>
  <Step>
    Generate the output.
  </Step>
  <Step>
    Review and refine the result.
  </Step>
  <Step>
    Save, copy, export, or send the result only when that action is visibly available.
  </Step>
</Steps>

## What to expect

Available actions may differ by prompt, account, or current product release. Use only the actions shown in the prompt you are working on.
