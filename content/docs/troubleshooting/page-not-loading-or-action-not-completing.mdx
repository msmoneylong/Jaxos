# Troubleshoot a Page or Action That Is Not Completing

## Symptom

A page remains blank, a loading indicator does not finish, a button appears not to respond, a generation or submission does not complete, or a page still looks outdated after an action.

## Likely causes

- The visible action is still in progress.
- Your internet connection is not currently available or stable.
- The page needs to be refreshed.
- The issue continues in the current browser session.

## Resolution steps

<Steps>
  <Step>
    Wait briefly for the visible action to finish.

    **Expected result:** You can see whether JaxOS updates the page or shows a visible result.
  </Step>
  <Step>
    Confirm that your internet connection is available.

    **Expected result:** You can continue using other online pages or services.
  </Step>
  <Step>
    Refresh the page once.

    **Expected result:** JaxOS reloads the current page.
  </Step>
  <Step>
    If the interface allows it and unsaved work is not at risk, sign out and back in.

    **Expected result:** You return to JaxOS with the account and context available to you.
  </Step>
  <Step>
    Retry the action once. If the issue continues, try a current supported browser or a private window when appropriate.

    **Expected result:** You can determine whether the action completes in a fresh browser session.
  </Step>
</Steps>

<Callout type="warn">
Do not repeatedly purchase, submit, publish, or generate while the result is uncertain. Check the visible result first, then record what happened if you need support.
</Callout>

## When to contact support

Contact support if the issue continues after these steps. Record the exact page, approximate time, action you attempted, and any visible message or status.

## Information to gather

- Page or feature name.
- Approximate date and time.
- The action you attempted.
- Any visible message or status.
- Whether the issue continued after one refresh or one retry.
- A safe screenshot with private information removed.
