# Create Content in Creative Studio

Creative Studio workflows can vary by content type, account, and current release. Follow the prompts and use only the actions that are visible to you.

## Steps

<Steps>
  <Step>
    Open **Creative Studio**.

    **Expected result:** You can see the Creative Studio options available to your account.
  </Step>
  <Step>
    Confirm the correct workspace or client context.

    **Expected result:** Your work is associated with the intended context.
  </Step>
  <Step>
    Choose an available content workflow or content type.

    **Expected result:** The workflow shows the details it needs.
  </Step>
  <Step>
    Provide the requested details.

    **Expected result:** The workflow is ready for the next available action.
  </Step>
  <Step>
    Generate or create the content when that action is available.

    **Expected result:** A result or draft is shown for review.
  </Step>
  <Step>
    Review the result.

    **Expected result:** You can check whether the output is suitable for its intended use.
  </Step>
  <Step>
    Use only the actions currently shown, such as copy, save, download, schedule, or send, when available.

    **Expected result:** JaxOS completes the action you selected when it is available for that item and account.
  </Step>
</Steps>
