# Open Creative Studio

Use this guide when you need to begin an available Creative Studio workflow or arrive from another JaxOS area.

## Steps

<Steps>
  <Step>
    Open **Creative Studio** from navigation when it is available.

    **Expected result:** The Creative Studio area opens for the context currently shown in JaxOS.
  </Step>
  <Step>
    If you arrived from another JaxOS workflow, confirm that you are in the intended Creative Studio experience.

    **Expected result:** You can see the available content workflows or content types for that path.
  </Step>
  <Step>
    Confirm the correct workspace or client context before continuing.

    **Expected result:** You are ready to work with content for the intended context.
  </Step>
</Steps>

## If Creative Studio is locked or unavailable

Creative Studio access depends on account type and permissions. Confirm the current workspace or client context, then use the navigation available to your account. If you still cannot access the area you expect, contact support.
