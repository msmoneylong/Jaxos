# Request Changes to Content

Request changes when content needs revision before it is suitable for its intended use. Use the available option for requesting changes only when it is available to your account.

## Write useful feedback

Be specific about the exact wording, image, date, link, offer, or factual issue that needs attention. Explain the desired correction so the next reviewer can understand what to change.

Avoid vague feedback such as “fix this” or “make it better.” Instead, identify the part that needs revision and the outcome you need.

## What to expect

After you submit the request, JaxOS records the response shown for the item. Review the visible status and item details for the current state.
