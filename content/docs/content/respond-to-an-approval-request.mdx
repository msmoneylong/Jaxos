# Respond to an Approval Request

Some Content items require review before the next available action can be taken. Approval authority is not available to every account.

## Respond to a request

1. Locate an item that is awaiting review in the Content area available to you.
2. Open the full item and inspect the content, media, date, destination, and notes shown.
3. Choose an available approval response only after completing your review.
4. Submit the response when JaxOS presents that option.

After you submit a response, the item shows the result currently recorded in JaxOS. Review the visible item status for the current state.

## If you do not see approval options

Approval actions depend on permissions and item status. Confirm the correct workspace or client context and use the options shown for your account. Contact support if you expected approval authority but do not have it.
